动态规划

发表于 2016-08-12   |   分类于 复习记录

动态规划的思想就是每次都是最优，综合到了最终得到的方案也是最优。是比较常用的算法，归结起来是以下这几个：

  1. 背包问题
  2. 记忆化
  3. Warshall算法
  4. Floyd算法
  5. 最优二叉查找树

背包问题

有n个物体有重量和价值两个属性，一个能承重一定重量的背包。问怎么选择物体能实现背包里的价值最大化。

假设有5个物体和一个背包。物体的重量分别是2、2、6、5、4，即W[]={0、2、2、6、5、4}，价值分别是6、3、5、4、6，即V[]={0、6、3、5、4、6}。背包承重为10。问怎么选择，能实现背包所背物体价值的最大化。

背包的状态转换方程 f[i,j] = Max{ f[i-1,j-Wi]+Vi( j >= Wi ), f[i-1,j] }
f[i,j]表示在前i件物品中选择若干件放在承重为 j 的背包中，可以取得的最大价值。
Pi表示第i件物品的价值。
